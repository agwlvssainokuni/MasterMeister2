# Story Generation Plan（変更要求: 接続コンテキストのグローバル化 + クエリ実行時スキーマ指定）

## 方針
既存の`aidlc-docs/inception/user-stories/personas.md`（管理者／一般ユーザ）をそのまま再利用する
（新しいユーザ種別は生まれないため、新規ペルソナ策定は行わない）。`requirements.md` §9.2で
確定済みの決定事項を、既存の`stories.md`の該当ストーリーの改訂＋新規ストーリーの追加という形で
反映する。
